# PerformaScan
### A Lightweight Benchmarking Tool for Rust Applications

## Description
PerformaScan is a simple, easy-to-use benchmarking tool built with Rust. It's designed to help developers measure the performance of their applications, identify bottlenecks, and optimize code for better efficiency.

## Features
* **Easy benchmarking**: Run benchmarks with minimal setup and configuration.
* **Customizable**: Create custom benchmark tests tailored to your specific use case.
* **Detailed reports**: Get in-depth insights into your application's performance, including execution time and memory usage.
* **Extensible**: Easily integrate PerformaScan into your existing Rust projects.

## Installation and Usage
### 1. Build PerformaScan
To build PerformaScan, run the following command in your terminal:
```bash
cargo build --release
```
### 2. Run PerformaScan
Once built, you can run PerformaScan using:
```bash
cargo run
```
This will execute the benchmark tests and display the results.
